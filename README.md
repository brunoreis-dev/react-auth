# Rocketseat - React Auth

## Clonando o projeto

`| git clone git@github.com:brunoreis-dev/react-auth.git |`

## Instalação

`| yarn install |`

## Executando o projeto

Abaixo os comandos que são usados no projeto:

| Comando | Description |

| ----------------------- | --------------------------------------------------------------------------------------------- |

`| yarn install |` Instala as dependências do projeto |

`| nextauth - yarn dev |` Executa o projeto em modo de desenvolvimento na porta `3000` |

`| auth backend - yarn dev |` Executa o servidor em modo de desenvolvimento |
